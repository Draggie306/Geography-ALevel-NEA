Word Cloud Generator

This web interface has beem made in 20 minutes so it's not the prettiest.
Enter the words below, can be comma separated if you want
Then click generate and it will generate a word cloud for you.
You can split the words up with commas, spaces, new lines or any combination of the three, the code should be smart enough to figure it out.
It will also remove any punctuation and numbers.

Open-source project available on GitHub!
